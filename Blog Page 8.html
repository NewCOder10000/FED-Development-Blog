Intro to Html, Css and JavaScript



  • In Week 08: I learned about Data Attributes. Data attributes, often referred to as "data-*" attributes,
    are custom attributes that can be added to HTML elements to store extra information.
    These attributes are prefixed with "data-" followed by a descriptive name that describes the information they contain.

    I also learned how to work with api. Examples of api are AJAX and RestDB.
    API stands for Application Programming Interface. It is a set of rules, protocols,
    and tools that allows different software applications to communicate with each other.
    APIs define the methods and data formats that developers can use to
    interact with a particular software component or service.
