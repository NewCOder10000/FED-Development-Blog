  • In Week 12: I learned RestDB and NoSql. NoSql is NoSQL, which stands for "Not Only SQL,"
    is a term used to describe non-relational databases that provide a mechanism for storage and retrieval of data.
    Unlike traditional relational databases (such as MySQL, PostgreSQL, or Oracle),
    NoSQL databases do not rely on the tabular schema of rows and columns.
    Instead, they use flexible data models that can be optimized for specific use cases.

    RestDB is a cloud-based NoSQL database service that provides developers with a platform
    for building and managing RESTful APIs and databases. It allows users to create custom databases
    with JSON-based data models, define endpoints for interacting with the data via HTTP
    methods (GET, POST, PUT, DELETE), and secure access to the data using authentication and
    authorization mechanisms.
