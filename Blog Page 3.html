  • In Week 03: I learned about advanced css layouts such as flexboxes.Flexbox, short for "Flexible Box Layout,"
    is a CSS layout model that provides an efficient way to design and distribute space within a container
    and align items within it.It is good for creating complex layouts and aligning items
    within a container or web page. You can align Flexboxes with css codes such as justify content,
    this adjusts the alignment, spacing, and distribution of the flexboxes, another example is
    flex direction, with this you can define the direction the flex items are layed out in the container
    with the row and column option. Another thing i learned in Week 03 is that !important is used to give a
    css rule higher priority, allowing it to override other rules.For oal, i just practiced what i have
    learned this week.
