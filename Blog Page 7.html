Intro to Html, Css and JavaScript



  • In Week 07: I learned about DOM. DOM stands for Document Object Model. It is a programming interface for web documents.
    It represents the structure of a document, such as HTML or XML,
    as a tree structure where each node represents a part of the document (e.g., elements, attributes, text).
    The DOM provides a way for programs to manipulate the structure, content, and style of web documents dynamically.

    I also learned about Json Local Storage, which refers to storing data in LocalStorage in JSON format.
    This means that data is serialized to JSON format before being stored in LocalStorage and
    deserialized back to its original format (e.g., JavaScript object) when retrieved from LocalStorage.
    This is a common practice for storing structured data in web applications,
    allowing for easy manipulation and retrieval of data within the client-side JavaScript code.
