Intro to Html, Css and JavaScript



  • In Week 02: I learned more html tags such as the nav tag, the article tag the section tag and the footer tag. The nav tag
    is used to define a part of a web page that contains links or menus, this will help to structure the content
    of a webpage. The article tag is used to define a independent piece of content
    within a web page. The section tag is used to define logically related section of a web page
    and the footer tag is used to define the footer of a webpage. I also learned about coding with css and how
    to link a css file to a html file, and how to style a html file with css, like how to change the background
    or foreground colour of a webpage, aligning text to different parts of a webpage and changing the font of the text
    of a webpage. One way of changing the font is by importing fonts from Google Fonts.For OAL i just learning more
    about these tags and their functions.
